## 🧩 What is Omni?

Omni is a desktop app designed to help you interact with multiple messaging platforms and AI tools in one place. It connects to over 25 different messaging services, works safely with sandboxed plugins, and keeps your content secure. Omni runs on Windows and is built using Rust and Tauri, which makes it fast and reliable.

The app helps you manage conversations, automate tasks, and access AI capabilities without switching between multiple apps. It also supports WebAssembly plugins that add new features without risking your data or system safety.

## 🔐 Security and Privacy

Omni uses several steps to keep your data safe:  

- Plugins run in a sandboxed environment, isolating them from your system.  
- The app limits access to sensitive files and information.  
- Your messages stay private unless you explicitly share them through integrations.  
- Omni is built on Rust, which helps reduce software bugs and vulnerabilities.  

These features protect you from common security issues found in other AI or messaging apps.
